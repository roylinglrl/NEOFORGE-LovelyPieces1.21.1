Lovely Sparkle Pieces 为 Minecraft 游戏添加了一系列独特的配件，旨在增强梦幻般的游戏体验

主要内容：

大量可穿戴的奇珍异宝均围绕 curiosAPI 制作。

从提升机动性的靴子到增强战斗力的戒指，各种精美的工具，以及强大的套装内容

设置效果：

该模块新增套装效果。 当前版本仅有一个套装：赌徒。

集齐3/5套套装后，即可获得额外效果。

套装效果通常相互关联。
